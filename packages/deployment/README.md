This package now requires the installation of `safe-hash-rs` (from Cyfrin) as a CLI tool which can installed using
the instructions below.

# Installation

## Curl

```shell
curl --proto '=https' --tlsv1.2 -LsSf https://github.com/Cyfrin/safe-hash-rs/releases/latest/download/safe-hash-installer.sh | sh
```

## npm

```shell
npm install -g @cyfrin/safe-hash
```

Now you can use the script from anywhere by just typing `safe-hash`.
